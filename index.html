Quick Reference Commands

Handy commands that don't fit into the categories above but come up constantly in everyday use.

Session & Housekeeping

reinitialize                  # full reset — clears everything
delete all                    # remove all objects but keep settings
save session.pse              # save full session
load session.pse              # restore a saved session
undo                          # undo last action (limited)
quit                          # exit PyMOL

Display & Background

bg_color white                # white background (default is black)
orient                        # align molecule along principal axes
reset                         # reset view to original orientation
set seq_view, on             # show the sequence viewer bar
set valence, 1               # show double/triple bonds
set depth_cue, 0             # disable distance fog

Exporting Images

ray 2400, 2400               # ray-trace at high resolution
png figure.png, dpi=300     # save image after ray tracing
png figure.png, ray=1       # ray-trace and save in one step
set ray_opaque_background, 0 # transparent background for export

Structural Alignment

align obj1, obj2             # sequence-based alignment (best for >30% identity)
super obj1, obj2             # structure-based (better for low similarity)
cealign obj2, obj1           # CE algorithm (most robust, but target comes FIRST)
rms_cur obj1, obj2           # RMSD without fitting (current coordinates)

Object Management

enable obj1                  # show object in viewer
disable obj1                 # hide object from viewer
group mygroup, obj1 obj2     # group objects together
create newobj, chain A       # create new object from selection
extract ligand_obj, organic  # extract selection into new object (removes from original)

Scripting Basics

@myscript.pml               # run a .pml script
run myscript.py              # run a Python script
log_open mylog.pml           # start recording commands to a file
log_close                     # stop recording

The Morph Function

The morph command (Incentive PyMOL) uses the RigiMOL algorithm to generate smooth interpolations between two conformational states, preserving local geometry better than simple linear interpolation. It's ideal for visualizing open/closed states, ligand-induced changes, and domain movements.

🧬
Structure A
Open / Apo state
→
✦✦✦
Interpolated
Frames
→
🔬
Structure B
Closed / Holo state

Command Syntax

morph mymovie, obj1, obj2, refinement=3, steps=30, method=rigimol

Parameters Reference

ParameterDescription
steps NNumber of intermediate frames. Default: 30
refinement NSculpting cycles to clean distorted intermediates. Default: 3
method rigimol|linearrigimol preserves local geometry (default). linear = simple Cartesian interpolation
match align|super|in|likeHow to pair atoms between the two structures
state1 0Morph between all consecutive states in a multi-state object (loop)
💡 GUI shortcut You can also access the morph from the Object Panel: select your object, then click A → generate → morph → to molecule and pick the target object.

Morph Walkthrough

This walkthrough uses Adenylate Kinase (1AKE → 4AKE) as the working example.

1

Fetch the Structures

fetch 1akeA, async=0
fetch 4akeA, async=0
2

Align the Structures

align 1akeA, 4akeA
3

Run the Morph

PyMOL creates a new multi-state object containing the interpolated trajectory.

morph mymovie, 1akeA, 4akeA, refinement=3, steps=30
4

Hide Originals & Style

disable 1akeA
disable 4akeA
as cartoon, mymovie
util.cbc mymovie
5

Play the Morph

Use the VCR controls at the bottom of the Viewer, or run:

mset 1 x30
mplay
6

Export as Video (Optional)

set ray_trace_frames, 1
mpng morph_frames/frame
# Then stitch with FFmpeg:
# ffmpeg -r 30 -i frame%04d.png -c:v libx264 output.mp4

Useful Utility Commands

frame 15                     # jump to frame 15
mstop                         # stop playback
delete mymovie               # delete morph object
undo                          # undo last command
help morph                   # open morph docs
